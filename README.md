# 📚 نظام إدارة المكتبة الذكي

تطبيق شامل لإدارة المكتبات مكتوب بلغة **C#** مع واجهة **Windows Forms** احترافية وسهلة الاستخدام.

## ✨ المميزات الرئيسية

### 1️⃣ إدارة الكتب
- ✅ إضافة كتب جديدة مع جميع التفاصيل (العنوان، المؤلف، ISBN، السنة، الفئة، الناشر، السعر)
- ✅ تعديل بيانات الكتب الموجودة
- ✅ حذف الكتب
- ✅ البحث عن الكتب بالعنوان أو المؤلف أو ISBN
- ✅ عرض عدد النسخ المتاحة من كل كتاب
- ✅ تتبع عدد النسخ الإجمالي والمتاح

### 2️⃣ إدارة الأعضاء
- ✅ تسجيل أعضاء جدد (الاسم، البريد، الهاتف، العنوان)
- ✅ تحديد نوع العضوية (فضي، ذهبي، بلاتيني)
- ✅ تعديل بيانات الأعضاء
- ✅ تعطيل أو تفعيل حساب العضو
- ✅ البحث عن الأعضاء بالاسم أو البريد أو الهاتف
- ✅ عرض قائمة كاملة بجميع الأعضاء

### 3️⃣ إدارة الإعارات
- ✅ إعارة الكتب للأعضاء
- ✅ تحديد تاريخ الاستحقاق تلقائياً (14 يوم)
- ✅ إرجاع الكتب
- ✅ حساب الغرامات للكتب المتأخرة (5 ريال لكل يوم)
- ✅ تتبع الإعارات النشطة والمرتجعة
- ✅ عرض تفاصيل الإعارة لكل عضو

### 4️⃣ التقارير والإحصائيات
- 📊 عرض إحصائيات المكتبة
  - إجمالي عدد الكتب
  - إجمالي عدد الأعضاء
  - عدد الإعارات النشطة
  - عدد الكتب المتاحة
- 📊 قائمة الكتب المتأخرة مع الغرامات المستحقة
- 📊 إجمالي الغرامات المستحقة

## 🚀 كيفية التشغيل

### المتطلبات
- .NET 6.0 أو أحدث
- Visual Studio 2022 أو Visual Studio Code

### خطوات التشغيل

#### الطريقة 1: استخدام Visual Studio
1. افتح Visual Studio
2. اختر `Open a project or solution`
3. انتقل إلى مجلد المشروع وافتح `LibraryManagement.csproj`
4. اضغط على `F5` لتشغيل المشروع

#### الطريقة 2: استخدام سطر الأوامر
```bash
# استنساخ المستودع
git clone https://github.com/jamilsharaf123/-.git
cd -

# تشغيل المشروع
dotnet run
```

#### الطريقة 3: بناء ملف تنفيذي
```bash
dotnet publish -c Release -f net6.0-windows
```

## 🔧 خدمات الفئة LibraryService

### إدارة الكتب
- `AddBook()` - إضافة كتاب جديد
- `GetAllBooks()` - الحصول على جميع الكتب
- `SearchBooks()` - البحث عن الكتب
- `UpdateBook()` - تحديث بيانات الكتاب
- `DeleteBook()` - حذف الكتاب

### إدارة ال��عضاء
- `AddMember()` - تسجيل عضو جديد
- `GetAllMembers()` - الحصول على جميع الأعضاء
- `SearchMembers()` - البحث عن الأعضاء
- `UpdateMember()` - تحديث بيانات العضو
- `DeactivateMember()` - تعطيل حساب العضو

### إدارة الإعارات
- `BorrowBook()` - إعارة كتاب
- `ReturnBook()` - إرجاع كتاب
- `GetOverdueLoans()` - الحصول على الكتب المتأخرة
- `GetMemberLoans()` - الحصول على إعارات العضو
